# Todo 应用

这是一个简单的待办事项应用，作为 Claude Code 的练习项目。

## 项目内容

| 文件 | 说明 |
|------|------|
| `index.html` | 待办事项应用 |
| `.cnb.yml` | CI/CD 配置 |

## 功能特性

- 添加待办事项
- 标记完成/未完成
- 删除事项
- 统计信息（总计、已完成、待完成）
- 数据持久化（localStorage）

## 使用方法

直接用浏览器打开 `index.html` 即可使用。
